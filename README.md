# MIN - Что это? 

**MIN** - это сборка мессенджера MAX, сделанная с целью защиты от слежки.

## Что было сделано:
* Удалено более 20 ненужных разрешений, среди них: камера, микрофон, местоположение, администрирование Bluetooth, доступ к контактам, автозапуск, доступ к хранилищу, фоновая активность, управление блокировкой экрана, загрузка и синхронизация данных без ведома пользователя и другие.

* Замена адресов серверов аналитики на заглушку:
  * `sdk-api.apptracer.ru >>> 127.0.0.1`
  * `tracker-api.vk-analytics.ru >>> 127.0.0.1`

## Какие есть ограничения?
Их на самом деле в этом клиенте довольно много из-за отобранных разрешений. А именно: не работающие звонки, аудиосообщения, нет возможности отправлять и сохранять файлы, нет возможности отправлять геопозицию. В общем нормально работают на 100% практически только обычные текстовые сообщения (ну и группы, чаты, каналы соответственно). Не думаю, что для чатов учебных заведений может понадобится что-то большее.
